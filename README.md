## サイト説明
  このサイトはゲームの攻略情報などを共有したい人を対象とし、掲示板のサイトとして開発しました。
  主な機能として、掲示板立ち上げ、コメント投稿、固定コメント、アンケート機能があります。このサイトで、欲しい情報が近い掲示板を開いた時、固定コメント機能でぱっと見たときに欲しい情報が手に入りやすく、分からないことがあればすぐに質問やアンケートなどで意見を求めることができるように意識しながら開発しました。
